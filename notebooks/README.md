## Folder Contents
- 'ProjectDL.ipynb'
The main notebook used for:
  - Data preprocessing
  - Model training
  - Fine-tuning
  - Model evaluation

- 'main.py' 
  A Streamlit script used to run the Batik classification web application.

## How to use the app (main.py) on streamlit:
1. There will be a button where you can put an image file
2. After you inserted an image, there will be a preview of the picture
3. Scroll down, and click the "Prediksi button"
4. The result will be given

- Link to the streamlit: https://website-batik-classification.streamlit.app/
